# Tastipies

> Search, find create and share your favorite recipes with the entire world

- [Live Demo](https://tastipies.herokuapp.com)

## Introduction

Tastipies is a portfolio project written in Express and React. It isn't meant to be a production ready app, but can be tested and played with [here](https://tastipies.herokuapp.com).

The application lets you create, manage and share recipes. **Still in development**, might contain bugs or miss features.

## Technologies

The application is completely written in JavaScript and hosted with Heroku. Here's a list of technologies/frameworks I used:

**Backend**

- Node.js
- [Express](http://expressjs.com/)
- MongoDB ([mLab](https://mlab.com/welcome/))
- [Mongoose](https://mongoosejs.com/)
- JWT ([JSON Web Tokens](https://jwt.io/))

**Frontend**

- React
- SCSS
- [axios](https://github.com/axios/axios)
- [Reach Router](https://reach.tech/router)

**Testing**

- Mocha & Supertest (Backend)
- Jest (Frontend)
